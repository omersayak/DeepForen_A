## ‚ö†Ô∏è Troubleshooting

*   **"Permission Error" during scan:** Ensure you ran the backend with `sudo`.
*   **AI Error (429/Quota):** The system handles rate limits gracefully. If Gemini is busy, try switching to ChatGPT or wait a minute.
*   **Devices show "Unknown":** The scanner uses a "Static Fallback" list for common vendors (Hikvision, Apple, etc.). Ensure the device is powered on and reachable.
